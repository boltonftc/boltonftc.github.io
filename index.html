Welcome to Bolton Robotics FTC — a hands-on FIRST Tech Challenge program where students go from a box of parts to a driving mecanum robot in nine sessions. Use the navigation above or the quick links below to get started.

🤖 PIT Mode — Learn to Code Without a Robot

The FTC 3D Simulator (PIT Mode) is a custom Godot-based simulator that lets students learn to program an FTC robot before ever touching the real hardware. The PIT (Programming & Integration Terminal) puts you in a virtual workshop with a 3D robot, a Driver Hub, and a full Java code editor — all in one window.

Lessons in the simulator guide students through writing real Java code step by step. That same code compiles and runs in the simulator, and when you're ready, it transfers directly to your real robot — no rewriting required.

3D Robot PreviewOrbit a real-time 3D model of the robot on a virtual pit table.
Live Code EditorWrite Java right in the browser — syntax highlighting, line numbers, instant feedback.
Real TelemetrySee your telemetry.addData() output on the virtual Driver Hub, just like the real thing.
Motor & Servo SimulationFlywheels spin up with inertia, servos sweep to position — physics you can see.
Encoder & IMU SensorsRead RPM from the flywheel encoder, heading from the IMU — real sensor data from virtual hardware.
Full Match SimulationRun TeleOp and Autonomous OpModes through a simulated FTC match sequence.
