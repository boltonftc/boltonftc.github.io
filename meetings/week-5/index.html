Engineering Notebook — Intake Drawing

Start here. Sketch your intake design in your engineering notebook before building or coding anything. Include dimensions, where it mounts on the robot, and how pollen enters. Label the key parts.

Programming Lessons — Up Through MecanumDrive

Work through the programming lessons in order. Make sure you have a driving robot before moving on. Complete all lessons up through and including the MecanumDrive lesson.

Brainstorm & Prototype — Pollen Transfer into Hopper

How does pollen get from your intake into the hopper? Sketch your best ideas in the engineering notebook, then prototype the most promising one. Think about what mechanism moves the pollen — belt, roller, ramp, chute?

📖

New Lesson — Coding Servos (Positional Mode)

Once your drive code is running, try the servo lesson. It covers how positional servos work, how to wire and configure one, and how to code a toggle using edge detection and a state machine.
