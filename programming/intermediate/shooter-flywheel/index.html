Flywheel Shooter

The Flywheel Shooter

Your robot has a mechanism for scoring — a flywheel shooter. It works the same way a baseball pitching machine does: a heavy wheel spins up to high speed, and when a ball is pushed against it, the wheel's energy transfers to the ball and launches it toward the goal.

💡 Key Idea
A **flywheel shooter** stores rotational energy in a spinning

wheel. When a ball contacts the spinning surface, friction transfers energy to the ball and launches it.

How Flywheels Work

A flywheel is just a motor with a heavy wheel attached. When you apply power, the wheel slowly accelerates. Once it reaches full speed, all that spinning mass holds a large amount of kinetic energy — energy stored in motion.

The key physics concept here is inertia — an object's resistance to changes in its motion. A lightweight wheel has low inertia and spins up quickly, but it also loses speed quickly when a ball hits it. A heavier wheel has more inertia — it takes longer to spin up, but it barely slows down when launching a ball. That stability is what you want.

💡 Key Idea
**Inertia** is an object's resistance to changes in motion.

A heavier flywheel takes longer to spin up, but maintains speed better through each shot — giving you more consistent launches.

Think about a merry-go-round. A small, light one speeds up when you push it but slows down fast. A massive steel one is hard to get moving, but once spinning, even pushing against it barely changes its speed. That is high inertia — exactly what makes a good flywheel.

Why Inertia Matters for Shooters

When a ball contacts the spinning flywheel, friction grabs the ball and accelerates it. But physics says energy is conserved — the energy that goes into the ball has to come from somewhere. It comes from the flywheel slowing down.

If the flywheel is light (low inertia), it loses a lot of speed on each shot. Your first ball flies fast, but the second and third are slower because the wheel has not recovered yet. If the flywheel is heavy (high inertia), it barely notices the ball. Every shot comes out at nearly the same speed.

On a real FTC robot, teams add extra mass to the flywheel to increase inertia. Metal discs, steel rings, or weighted hubs bolted to the flywheel all work. More mass = more inertia = more consistent shots. The tradeoff is spin-up time — heavier flywheels take longer to reach full speed, so you need to plan when to start spinning.

>> Type This Code
// ── Flywheel Toggle ──────────────────────────────


// Get the shooter motor from the hardware map.
// This is the motor that spins the flywheel.
DcMotor shooterMotor = hardwareMap.dcMotor.get("shooter_motor");




// The base power level for the flywheel.
// 0.65 = 65% power. This is a good starting point
// that gives enough speed to score without wasting
// energy. You can adjust this later.
final double FLYWHEEL_BASE_POWER = 0.65;




// Check if the flywheel is currently spinning.
// If power is above 0.01, it is ON.
boolean isRunning = Math.abs(shooterMotor.getPower()) > 0.01;




// Toggle: if running, stop it. If stopped, start it.
shooterMotor.setPower(isRunning ? 0.0 : FLYWHEEL_BASE_POWER);

This is the same toggle pattern from the intake lesson:

  1. Get the motor from the hardware map
  2. Check if it is currently running (power > 0.01)
  3. If running, stop it; if stopped, start it at 65% power

Why 65% and not 100%? Full power on a flywheel is often too fast — balls can overshoot the target. 65% is a reasonable baseline that you can fine-tune later.

💜 Hint
**Pro tip:** On a real robot, spin up the flywheel a few seconds

before you plan to shoot. Do not wait until you are at the scoring position — the flywheel needs time to reach full speed. Many teams bind the flywheel toggle to a bumper so the operator can start it during approach.
